Open Aid Italia API are an applicative interface that allows any software module to access the data and are realised following REST architecture scheme.

The API access address is http://{{ request.get_host }}{% url 'api-root' %}.

Data are available in HTML and JSON formats.
The following data types are available:

  • Projects, the list of internation cooperation proyects
  • Project, the detail data on a single project
  • Activities, the list of activities related to a project realised during the years
  • Recipients, the codelist of countries and geographic zones that receive international help
  • Agencies, the list of Italian subjects that finance the projects
  • Sectors, the codelist of sectors of intervention related to the projects
  • Finance_types, type of possibile financing supplied
  • Channels, financing supply channel codelist
  • Aid_types, aid type codelist

For example:

  • Complete list of all projects: http://{{ request.get_host }}{% url 'project-list' format='json' %}
  • Detail of a single project: http://{{ request.get_host }}{% url 'project-detail' format='json' pk='PROJECT_ID' %}
  • List of sector of intervention: http://{{ request.get_host }}{% url 'sector-list' format='json' %}

All data are released with CC BY-SA 3.0 license.

Browsing the API with a browser a detailed documentation in English is available.
