L'API di Open Aid Italia, con architettura REST, è un'interfaccia applicativa che permette a qualunque componente software esterno di accedere ai dati.

L'indirizzo per accedere alla API è http://{{ request.get_host }}{% url 'api-root' format='json' %}.

I dati sono disponibili in formato HTML e JSON.
Sono a disposizione i seguenti dati:

  • Projects, elenco dei progetti di cooperazione intenazionale
  • Project, dettaglio dei dati su un singolo progetto
  • Activities, le attività negli anni legate a un singolo progetto
  • Recipients, la codelist dei Paesi e zone geografiche che ricevono gli aiuti e dove sono localizzati i progetti
  • Agencies, la lista dei soggetti del nostro Paese che finanziano i progetti
  • Sectors, la codelist dei settori a cui fanno riferimento i progetti
  • Finance_types, le tipologie dei finanziamenti erogati
  • Channels, la codelist dei canali dei finanziamenti
  • Aid_types, la codelist delle tipologie di aiuto

Alcuni esempi:

  • La lista di tutti i progetti: http://{{ request.get_host }}{% url 'project-list' format='json' %}
  • Il dettaglio di un singolo progetto: http://{{ request.get_host }}{% url 'project-detail' format='json' pk='PROJECT_ID' %}
  • La lista dei settori d'intervento: http://{{ request.get_host }}{% url 'sector-list' format='json' %}

Tutti i dati sono sono rilasciati con licenza CC BY-SA 3.0.

Navigando la API con un browser è possibile accedere ad una documentazione dettagliata in lingua inglese.
